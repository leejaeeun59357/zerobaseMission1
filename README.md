**프로젝트 주제** 
<br/>내 위치를 기반으로 서울시 공공 와이파이 정보를 제공하는 웹서비스 개발


**세부 소개** 
<br/>OPEN API를 이용하여 내가 서비스 하고자 하는 데이터로 마이그레이션 하여 웹을 기반으로 원하는 형태로 출력하기, 또한 사용한 데이터를 히스토리 데이터베이스에 저장하는 기능과 출력하는 기능 구현


**구현 기능**
1. OPEN API 가져오기
2. 나의 위치정보를 바탕으로 가까운 위치에 있는 와이파이 정보 20개만 출력하기
3. 내가 입력한 위치 정보에 대해서 DB에 히스토리로 저장 및 출력하는 기능
4. 와이파이 정보에 대한 북마크와 북마크 그룹 생성 및 출력하기 기능
